## Purpose

The Network page has been designed so all maps and route graphics are independent of the website layout.

This allows new destinations to be added without rebuilding the page.

Future versions will include:

- Interactive route map
- Animated route lines
- Airport information pop-ups
- Fleet filtering
- Long-haul expansion
